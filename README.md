# 3D Print Models

A collection of 3D printable models in STL format, organized by category.

## Repository Structure

```
models/
├── basic-shapes/    # Fundamental geometric shapes (cube, sphere, cylinder, etc.)
├── home/            # Household items and organizers
└── tools/           # Practical tools and fixtures
previews/            # Preview images of models
```

## Getting Started

1. Browse the `models/` directory to find a model you want to print.
2. Download the `.stl` file.
3. Open it in your slicer software (e.g., PrusaSlicer, Cura, Bambu Studio).
4. Slice and print!

## Print Settings

Most models in this repository are designed with the following default settings in mind:

- **Layer Height**: 0.2 mm
- **Infill**: 15–20%
- **Supports**: None (unless noted)
- **Material**: PLA

Individual model READMEs may specify different recommended settings.
